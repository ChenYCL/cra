# `@lgoweb/cra`

🏗 Personal CLI to bootstrap new projects

## Installation & Usage

```bash
# or
npx @lgoweb/cra
# or
npm install -g @lgoweb/cra

cra -i # init prpject
cra -v # check version
```

## args

`-g`: 'git init'

`-y`: 'skip the promet'

`-i`: 'install dependencies'

`.eg: cra -i`
